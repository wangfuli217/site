** 前端开发经历

- 2014 之前 - 随便玩玩 :: WordPress 主题（静态页面） => _CSS_ / _HTML_
- 2014 ~ 2017 - 生产化（数据统计平台，ToB） :: 前后端不分离（SSR） => JS/CSS/HTML， [[https://getbootstrap.com/][bootstrap]]，[[https://webpack.js.org/][webpack]]
- 2018 ~ 2019 - 企业中后台 :: 前后端分离（SPA，CSR） => ES6/ _LESS_ /HTML，[[https://pro.ant.design/docs/getting-started-cn][Ant Design Pro]]（React 全家桶 + antd）

** 总结
