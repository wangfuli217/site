对 Emacs 的定制目标：

- *原生* 减少 package，能用自带的方案解决就使用自带的方案
- *实用* 不用华而不实的 package
- *稳定* 使用稳定为第一要务，不追求新
- *高效* 合理的编排快捷键，减少按键且易于记忆

配置文件： [[https://github.com/zhangjie2012/_emacs.d][https://github.com/zhangjie2012/_emacs.d]] 。
