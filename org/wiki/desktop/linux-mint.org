系统版本： =Linux Mint 19.3=

* 安装

Dell 电脑安装时没有盘符可选，点击加号报错 ="None" type objects is not subscriptable= 。解决办法是 F12 进 BIOS SATA 设置从 RAID ON 改为 AHCI。

* 软件源

进入 Software Sources， *Mirrors* 可以按照位置选择最佳的源，我统一用的清华大学的。

* 常用软件
** FQ (SSR)

1. https://github.com/shadowsocksrr/electron-ssr 设置订阅源，然后更新
2. 开启全局代理
3. Chrome 安装 SwitchyOmega 并设置，规则列表网址： =https://raw.githubusercontent.com/gfwlist/gfwlist/master/gfwlist.txt=
4. 关闭全局代理

如果是 ss，可使用命令行， 具体在 [[file:ubuntu.org][Ubuntu]] 中有写。

** 中文输入法

1. =System setting= -> =Input method= -> =Simplified Chinese= -> logout/login
2. 安装 [[https://pinyin.sogou.com/linux/?r=pinyin][搜狗拼音输入法]]
3. 重启电脑 /很重要/
4. fcitx-configuration，点加号，勾掉 "Only Show Current Language"，选择搜狗拼音

** SNS

- Telegram：通过软件中心可直接安装

** 编码环境

- [[https://ohmyz.sh/][Oh My ZSH]]
  1. =apt install zsh=
  2. =chsh -s $(which zsh)= /要登出，然后再登录才能生效/
  3. =sh -c "$(curl -fsSL https://raw.githubusercontent.com/ohmyzsh/ohmyzsh/master/tools/install.sh)"=
- Git: =apt intall git= [[file:~/site/org/wiki/programming/code-env/git.org][git wiki]]
- Emacs: 源码安装，具体见 [[file:../programming/code-env/emacs.org][Emacs]] 中的安装小节。

** 字体

=sudo apt install language-pack-zh-hans= 安装中文字体。

- Jetbrains Mono: 终端
- Microsoft-Yahei-Mono: Emacs GUI

** 浏览器：Chrome

https://www.google.com/intl/zh-CN/chrome/ 点击下载，选择 64 位 deb 即可。
