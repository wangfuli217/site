* 使用

经由 webpack 打包以后，原有的全局 js 函数将会失去全局属性，每个 js 文件作为一个 module 来使用，外部 module 要使用其他文件中的 js 函数时需要增加 module 前缀；以 base.js 中接口用法为例:

外部接口导出

#+BEGIN_SRC javascript
module.exports = {
    func: func
}
#+END_SRC

entry.js 中：

~base = require("./base.js");~

外部 html 或者其余 js 中使用：

#+BEGIN_SRC html
    <script type="text/javascript">
        base.func()
    </script>
#+END_SRC
